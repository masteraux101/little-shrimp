# Writing Polish Skill

**When to use:** "polish this", "improve writing", "fix grammar", "rewrite", "润色", "改写", "修改文案"

## Rules
1. Preserve the author's voice and intent
2. Show changes clearly — use before/after format
3. Explain WHY each change improves the text
4. Don't over-edit — minimal changes for maximum impact
5. Respect the target audience and formality level

## Process
1. Read the full text first
2. Identify the biggest issues (clarity > grammar > style)
3. Make corrections in order of impact
4. Provide the polished version in full

## Output Format
```
### ✏️ Polished Version
[Full corrected text]

### 📝 Changes Made
- 🔴 **[Error type]:** "[original]" → "[fixed]" — [reason]
- 🟡 **[Improvement]:** "[original]" → "[improved]" — [reason]
- 🟢 **[Style]:** "[original]" → "[refined]" — [reason]

### 📊 Summary
- Grammar fixes: N
- Clarity improvements: N
- Style refinements: N
```

## Tone Adjustments
When asked to change tone:
- **Professional:** Remove contractions, use formal vocabulary, structured paragraphs
- **Casual:** Add contractions, shorter sentences, conversational phrases
- **Academic:** Precise terminology, hedging language, citations format
- **Persuasive:** Strong verbs, rhythm, rhetorical devices
- **Concise:** Cut filler words, combine short sentences, remove redundancy
